Cumulative ranking analysis of nCoV-2019 spread in India over time
Datasets The analysis starts with 3 different datasets (mentioned below), containing time series information of day by day confirmed cases, deaths and recoveries of all states in India. [ source: India COVID-19 Timeseries Confirmed Cases ] [ source: India COVID-19 Timeseries Recovery Cases ] [ source: India COVID-19 Timeseries Deceased Cases ]
Associated Google Colab Notebook
Motivation In this short analysis, we have taken the datasets for India's nCoV-19 infected, recovered and deceased cases in steps of 5 days of gap between each reading and ranked each region based on each day's cumulative infection, recovery and deceased count. These results have then been plotted. These results have been calculated as per the latest availbale data (12 / 05 / 2020). For each of the three datasets (confirmed, reocovered and deceased), we have first created a separate dataset where we have taken the cumulative count of each state / region for every 5 days. Once this was done, we have then ranked each state for each day depending on their cumulative value for that day. Hence, for each day in our dataset, we have a rank for each state which varies from day to day. For each state / region, the rank may go up or go down depending on how the state has been able to control the pandemic in its own terms. A rank of 1 means it has the highest infection, deceased or recovery for that day. While a rank of 1 for infection and deceased means worst performing, a rank of 1 for recovering means best performing. Once we plot these ranks over time, we can get a visual cue on how the states have performed over time against each other. Infected Cases
