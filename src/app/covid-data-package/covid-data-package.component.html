The  covdata  package
PyPI github
This package has the power to bring all data of India as tabular form up-to previous day you are working. This module will bring the whole set of raw data in a date-wise and processed format from the COVID-19 Indian Dataset maintained by University of Kalyani, which will be easier to deal with. Once installed, we can get an up-to-date information about the Covid-19 affected people, till the day the data is available at the source. We can visualize the data of our choice for any given range of dates. It is capable of giving information about the total number of confirmed, recovered and deceased patients, between any two dates, of any city, district or related to any particular gender (male/female). We can also visualize the whole up-to-date data without using any aforesaid filters using our module. But due to some unavailability of prefect data the state wise data sets are up to date. It can't be filtered by date. Beside that it will bring the whole raw data, date wise data. It can also visualize the data of your choice of range of dates. You can also see the total confirmed, death and recovered by filtered the data set giving the date or range of dates. Let's look at a small tutorial on how to use this package. STEP 00: download the package from PyPI. To do this, simply use pip install covdata STEP 01: import the package and initialize the module. To do this, you can use from covdata.covidindia import * init = initializer()
STEP 02: We can then see a data collection of our choice, such as "confirmed", "recovered" and "deceased". We can do this by calling the show_data method on the init object. init.show_data(of='recovered')
Good thing is we can also use this directly from our CLI. To do this, run the following command on your Terminal or Console: covdata -f Total_Recovered --options show_data
STEP 03: To perform data filtration, we should initiate the Data class and then call the appropriate methods on this instance. Let's get a dataframe of confirmed cases between two dates. filter = Data(init) filter.get_count_between_date('10/04/2020', '14/04/2020', by='total confirmed')
This can also be achieved from the command line interface. covdata -f Total_Confirmed -d 10/4/2020-14/4/2020 --options count_between_date
Step 04: We can also get the overall visualization of daily counts or cumulative counts. If daily argument is not mentioned, the cumulative graph will be shown. graph = visualizer(init) graph.whole(daily=True)
This tutorial shows a basic usage of this powerful package. To know more, you can visit the GitHub or PyPI link or view its documentation.
